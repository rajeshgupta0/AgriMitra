# 🌾 AgriMitra AI

**AgriMitra AI** is a Voice-First Smart Farming Assistant designed to empower Indian farmers with AI-powered crop guidance, disease detection, weather intelligence, market insights, and government scheme recommendations in their local language.

## 🚀 Features

### 🎙️ Voice-First AI Assistant

* Multilingual voice interaction
* Speech-to-Text (STT)
* Text-to-Speech (TTS)
* Farming-related AI guidance

### 🌱 Crop Disease Detection

* Upload crop images
* AI-powered disease identification
* Treatment recommendations
* Prevention guidelines

### 🌦️ Weather Intelligence

* Local weather updates
* Rainfall predictions
* Farming activity recommendations

### 📈 Market Insights

* Crop price monitoring
* Sell/Wait recommendations
* Market trend analysis

### 🏛️ Government Schemes

* Latest agriculture schemes
* Farmer eligibility information
* Application guidance

### 👨‍🌾 Farmer Profile Management

* Farmer onboarding
* Crop information
* Land details
* Irrigation details

### 📱 Progressive Web App (PWA)

* Offline support
* Installable on mobile devices
* Fast loading experience

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/rajeshgupta0/AgriMitra.git
cd AgriMitra
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
